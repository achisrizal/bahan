Uji Perbedaan Kecepatan Loading Gambar

Gambar Besar (Ukuran File Besar)

Gambar ini memiliki ukuran file yang lebih besar dan akan memakan waktu lebih lama untuk dimuat.

Gambar Kecil (Ukuran File Kecil)

Gambar ini lebih kecil dan akan dimuat lebih cepat dibandingkan gambar besar.

Perbandingan ini hanya untuk menguji bagaimana gambar dengan ukuran file yang berbeda mempengaruhi kecepatan loading halaman.
