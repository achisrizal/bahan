Uji Perbedaan Kecepatan Loading Gambar (JPG, PNG, WEBP)

Gambar JPG (Ukuran File Lebih Besar)

Gambar ini berformat JPG dan memiliki ukuran file yang relatif besar.

Gambar PNG (Ukuran File Lebih Besar dan Tanpa Kompresi)

Gambar ini berformat PNG dan cenderung lebih besar karena tanpa kompresi lossless.

Gambar WEBP (Ukuran File Lebih Kecil dan Terkompresi)

Gambar ini berformat WEBP, yang menawarkan kompresi lebih baik dan ukuran file lebih kecil.

Perbandingan ini hanya untuk menguji bagaimana format gambar yang berbeda mempengaruhi kecepatan loading halaman.
